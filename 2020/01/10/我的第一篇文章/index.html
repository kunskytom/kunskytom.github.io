第一个博客

2020-01-10

##基于hexo框架的个人博客

##配置环境变量
先配置git环境变量，下载node.js。去官网下载即可。
用 node -v npm -v git version 查看版本

##
第二步安装镜像语言指向淘宝
npm install -g cnpm –registry=http://registry.npm.taobao.org

第三步安装hexo(都是全局安装)
cnpm install -g hexo-cli
hexo -v 检查

第四步创建blog文件夹
mkdir blog

第五步进入blog文件夹
cd blog/
然后初始化博客 hexo init

第六步启动hexo
hexo s

hexo n “ssss”(创建新文章)

hexo clean (清理)

hexo g(生成)

再重新启动

##将博客部署到github上

新建一个仓库 注意仓库名必须为用户名.github.io格式

然后在安装插件
cnpm install –save hexo-deployer-git

然后打开_config.yml文件配置内容
type: ‘git’
repo: 复制仓库地址
branch: master

hexo d (部署)
这里我发生了两次错误，第一次是因为配置文件内容的时候 type里的git的单引号不能删除。然后部署的时候还是错误，然后需要设置一下
git config –global user.name “”
git config –global user.email “”
然后部署成功。

到这博客已经成功被推的远端，已经成功。

##换主题

git clone +克隆的地址.git themes/主题名

然后配置_config.yml文件，找到里面的themes,改为要换的主题名

最后再重新清理一下，生成，启动，再推到github远端。

结束！。
